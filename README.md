# SGEProyectoFinal
Proyecto final de:
  - Ismael Gomez
  - Martin Miroslavov
  - Nicolás Pla
  - Pablo Orbea
  - Pedro Llorente.

La aplicación permite:
 Visualizar grupos sin profesor en un momento determinado.
 Asignar profesores de guardia a dichos grupos.
 Insertar o eliminar grupos sin profesor y profesores de guardia.
